  ЧТО ТАКОЕ UAF
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Формальная теория того, как системы остаются живыми, адаптивными
и организованными. Любая стабильная система — клетка, мозг, рынок,
язык, теория — существует потому что поддерживает замкнутый контур:

          структура → информация → катализ → структура

Когда контур размыкается — система деградирует.
Когда замыкается — поднимается на следующий уровень организации.

  МАСТЕР-УРАВНЕНИЕ  (v5, единственное)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  dA_i/dτ = α_s · C_ij · A_j · (1−A_i)    ← TSV: социальный рост
           + α_l · Π_i  · PE_i · (1−A_i)   ← FEP: байесовское обучение
           + f   · (1 − A_i/A_c)            ← basal: минимальный метаболизм
           − δ   · (1 − 0.3·A_i)           ← decay: энтропийный распад

  Где:
    A_i    — closure_degree ∈ [0,1]  (главный параметр, всем управляет)
    C_ij   — каталитический вес соседа j (BA-топология, хаб = 2.69×)
    Π_i    — precision агента i (обновляется из prediction error)
    PE_i   — prediction error = насколько агент далёк от оптимума
    f      — basal floor (минимальная активность, не даёт A→0)
    δ      — decay (энтропийный распад)
    τ      — relational time (число взаимодействий, не часы)

  ТОЖДЕСТВО TSV = FEP:
    При Π_ij = α_s·A_j  и  PE_i = A_i
    TSV-член = FEP-член. Один механизм, два языка.
    Каждый социальный агент бессознательно делает то,
    что байесовский агент делает сознательно.

  НАХОДКА 1 — БИСТАБИЛЬНОСТЬ С НЕСТАБИЛЬНЫМ РАВНОВЕСИЕМ
  ───────────────────────────────────────────────────────
  При β=0.06, δ=0.010 система имеет ТРИ точки равновесия:

    A* = 0       — поглощающее (смерть)
    A* = 0.563   — НЕСТАБИЛЬНОЕ (λ=+0.0135) ← истинный TippingPoint
    A* = 0.805   — устойчивое (λ=−0.0170)   ← аттрактор жизни

  a_crit = 0.75 — НЕ особая точка системы.
  Истинный водораздел — A*_unstable = 0.563.
  TippingPoint = пересечение этого водораздела, не порог 0.75.

  НАХОДКА 2 — СЕДЛОВО-УЗЛОВАЯ БИФУРКАЦИЯ  δ* = 0.01117
  ─────────────────────────────────────────────────────
  При δ < δ*: бистабильность (два режима выживания и смерти)
  При δ > δ*: нет верхнего аттрактора → система всегда умирает
  Тип: saddle-node bifurcation

  Floor сдвигает δ*:
    floor=0.000 → δ*=0.01117  (baseline)
    floor=0.002 → δ*=0.01187  (+6.3%)
    floor=0.005 → δ*=0.01299  (+16.3%)

  НАХОДКА 3 — FLOOR ОПУСКАЕТ БАРЬЕР, НЕ ПОДНИМАЕТ СИСТЕМУ
  ─────────────────────────────────────────────────────────
  Каждые +0.001 floor → A*_unstable смещается вниз на ~0.030.
  floor=0.002 → водораздел с 0.563 падает до 0.503.
  Механизм: floor снижает эффективный decay →
  TSV начинает побеждать при большем δ.
  Это объясняет 4× расширение rescue window в EXP 025e/f.

  НАХОДКА 4 — КОЛЛЕКТИВНЫЙ ЭФФЕКТ BA-ТОПОЛОГИИ
  ─────────────────────────────────────────────
  Одиночный агент из A=0.25: net dA = −0.0064 → смерть.
  Группа N=60 из A=0.25: выживает → жизнь.

  Почему: хаб (C_ij=2.69) при A=0.25 имеет α_eff=0.215,
  net dA ≈ −0.0001 — практически равновесие у порога.
  Хаб удерживается, листья тянутся через TSV.
  Это коллективный механизм преодоления барьера.
  Невозможен в одноагентной модели.

  НАХОДКА 5 — МОСТ EXP 044 ЗАКРЫТ
  ─────────────────────────────────
  v3.1: corr(closure_F, TSV_A) = +0.089  ← две независимые системы
  v5:   corr(precision, mean_A) = +0.72  ← общая переменная Π_i
  Мост закрыт через архитектуру (не корреляцию метрик).

  КЛЮЧЕВЫЕ ПОНЯТИЯ
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  AdaptiveLoop (AL):
    Минимальная единица. Тройка (A, I, C) где:
    A = closure_degree ∈ [0,1]  — единственный параметр в уравнении
    I = info_content             — следует за A (не управляет)
    C = catalytic_power          — следует за A (не управляет)

  TippingPoint:
    Пересечение водораздела A*_unstable (≈0.563 при стандартных param).
    НЕ порог mean(A) > 0.75 — это только удобный маркер внутри
    верхнего бассейна.

  Уровни организации (через Jacobian λ, не пороги):
    L0  A < A*_unstable,  λ > 0   — хаос, система в бассейне смерти
    L1  A > A*_unstable,  λ < 0   — переход, система перешла водораздел
    L2  устойчивая, std(A) > ε    — когерентность, неоднородная
    L3  устойчивая, std(A) < ε    — интеграция, однородная  ← Q3 открыто

  NPG (Normalized Predictive Gain):
    NPG(M; D, B) = (L(B,D) − L(M,D)) / (L(B,D) + ε)
    NPG > 0: модель лучше baseline
    NPG ≤ 0: отклонить в домене D
    ВАЖНО: старый NPG (через variance tail) тривиально +1.000 — сломан.
    Честный NPG: через разницу свободных энергий F_model vs F_baseline.

  Basal floor:
    Прямой добавок к A каждый шаг: A += f·(1 − A/A_c).
    Это минимальный метаболизм — не патч, а механизм.
    floor* ≈ 0.001–0.003 (оптимум зависит от decay).
    При floor > floor_sat: накачивает к потолку, динамика исчезает.
